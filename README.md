### CLI usage
Install it via PPA.
```bash
sudo apt-add-repository ppa:hda-me/urlhda
sudo apt-get update
sudo apt-get install urlhda
```
Or copy [this script](https://raw.githubusercontent.com/cryptofuture/urlhda/master/script/urlhda) to `/usr/local/bin`
